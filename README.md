# NumPy

This repository contains solutions to basic NumPy programming exercises.

## Questions Covered

### 1. Array Creation using `np.array()`
- Create a 1D array with elements `[10, 20, 30, 40, 50]`
- Create a 3×3 2D array
- Print arrays, shape, and data type

### 2. Arrays of Zeros and Ones
- Create a 1D array of 8 zeros
- Create a 4×4 array of ones
- Create a 3×3 matrix of zeros

### 3. Using `np.arange()`
- Generate numbers from 0 to 20
- Generate even numbers from 10 to 50
- Generate numbers from 5 to 100 with a step of 5

### 4. Using `np.linspace()`
- Generate 10 equally spaced numbers between 0 and 5
- Generate 15 equally spaced numbers between -10 and 10

### 5. Random Number Generation
- Generate random numbers using `np.random.rand()`
- Generate a random matrix using `np.random.randn()`
- Generate random integers using `np.random.randint()`

### 6. Vector Operations
- Vector addition
- Vector subtraction
- Element-wise multiplication
- Dot product

### 7. Matrix Operations
- Matrix addition
- Matrix multiplication
- Element-wise multiplication

### 8. Matrix Properties
- Shape
- Dimension (`ndim`)
- Total number of elements (`size`)
- Data type (`dtype`)
- Minimum and maximum values

### 9. Matrix Statistics
- Generate a random 4×5 matrix
- Calculate:
  - Sum
  - Mean
  - Standard Deviation
  - Maximum value in each row

### 10. Simple Statistics Calculator
- Generate random numbers based on user input
- Display:
  - Mean
  - Median
  - Standard Deviation
  - Minimum value
  - Maximum value
- Reshape into a 2D array (if possible)
- Calculate row-wise sum
- Exception handling for invalid input
